# kwimy-splash

Plymouth splash theme for non-encrypted boots in Kwimy Linux

## Install path

- `/usr/share/plymouth/themes/kwimy-splash`

## Files

- `kwimy-splash.plymouth`: theme descriptor
- `kwimy-splash.script`: Plymouth script
- `logo-splash.png`: logo asset
- `throbber-*.png`: animation frames

## Package

Built from this directory via `PKGBUILD` and installed as the `kwimy-splash` package

## Notes

- Default theme selection is handled by the installer when encryption is disabled
